# Jackson Weaver – Data Science Portfolio

This repository contains a collection of my data science and machine learning projects, including competition work and independent projects.

The `kaggle_projects` directory includes projects developed for data science competitions and structured machine learning problems.

The `personal_projects` directory contains independent projects, including a Helldivers 2 API data collection pipeline and time-series analysis of game event data targeting player count for hourly play. 

I am also working on collecting data for washer/dryer info to analyze time-series to see if people are predictable for times and settings or just time. This would mean we can look into people who fall into certain times such as people in apartments sharing a washer/dryer with other apartments explaining their behavior of times because we have survivorship bias in this, for I am not keeping tabs on everyone who uses the washer/dryer.
